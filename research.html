Partial Research Interests

Machine Learning

This work designs a series of methods for supporting Machine Learning (ML) tasks over communications network. The proposed solutions aim to optimize data storage, data transmission, and inference computation for ML over Next Generation Networks in different layers. These solutions are highly favorable for time-dependent and loss-sensitive applications such as the URLLC service in high-speed environments.

Communication Network

This research focuses on optimization algorithms for Cellular Network. The research results address and model the fundamental issues in 5G/6G networks. E.g. higher system capacity, higher data rate, lower latency, and improved QoS compared to 4G system.

Network Applications

This study aims to show an automated visual inspection concept. The visual inspection separates the detection from the rule-based analysis process and applies machine learning technologies to adopt incoming uncertainties. The flexibility of the concept is demonstrated by testing the analysis process in a testbed.

Distributed System

This research covers both mobile and wireline scenarios and considers both hardware and software improvements. The solutions aims to reside on the cloud infrastructure and among end devices, and includes both data plane and control plane.
